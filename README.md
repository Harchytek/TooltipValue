# TooltipValue

This mod reads [TradersExtended](https://thunderstore.io/c/valheim/p/shudnal/TradersExtended/) configuration files and automatically applies custom prices to item tooltips in the game.

<p align="left"><img src="https://imgur.com/uWXXBJd.png" alt="TooltipValue" width="510"></p>

## Features

* Automatically displays item values in the tooltip, using Valheim’s native UI system just like base game items.
* Simply set up your items in [TradersExtended](https://thunderstore.io/c/valheim/p/shudnal/TradersExtended/) and the mod will automatically synchronize their values.
* Designed to be lightweight, the mod runs only during the initial loading phase and has zero impact on game performance.

## Installation
1. Install [BepInExPack Valheim](https://valheim.thunderstore.io/package/denikson/BepInExPack_Valheim/).
2. Place `TooltipValue.dll` in your `BepInEx/plugins` folder.
3. Launch the game and you will see the prices of the items for sale.
